Hybrid Recommendation System

This system is a hybrid recommendation system developed to provide movie and music recommendations based on user interaction and preference data.

The recommendation process combines SVD++ and KNN algorithms to improve recommendation accuracy.

Recommendation Methods

SVD++

SVD++ is a collaborative filtering algorithm that improves matrix factorization by considering implicit user interaction behavior.

K-Nearest Neighbor (KNN)

KNN identifies similar items based on similarity between user interactions.

Hybrid Recommendation

The final recommendation score is generated by combining SVD++ and KNN prediction scores using weighted hybrid scoring.

Datasets

MovieLens Dataset

MovieLens dataset is used for movie recommendation experiments containing movie rating interactions.

Last.fm HetRec 2011 Dataset

Last.fm dataset is used for music artist recommendation based on listening behavior and user interaction data.

Evaluation Metrics

The recommendation system performance is evaluated using RMSE and MAE metrics to measure prediction accuracy.
